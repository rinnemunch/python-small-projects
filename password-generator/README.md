# 🔐 Password Generator

A simple, beginner-friendly Python app that creates secure passwords based on your preferences.

## ✅ Features

- Choose your desired password length
- Optionally include:
  - Uppercase letters
  - Numbers
  - Symbols
- Input validation to prevent invalid length or empty character sets

## 🚀 How to Run

```bash
python main.py
```
